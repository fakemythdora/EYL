# Make-bot-online-24-7
Dùng pack Express để lm bot online :))

Bạn copy cái code đó vào `index.js`, sau đó bạn lên [UptimeRobot](https://uptimerobot.com/) để host. Click vào Add New Monitor, click vào Monitor Type và chọn HTTP(s). Điền tên vào Friendly Name và dán cái web vào URL(or IP), tiếp theo bạn nhấn Create Monitor và xong (đợi tầm 1-2 phút để nó load data)!
